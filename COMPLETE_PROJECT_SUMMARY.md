## 🚀 Quick Start

### 1. Open Frontend
```bash
# Direct file access
open frontend/index.html

# Or use local server
python -m http.server 8080
# Visit: http://localhost:8080/frontend/
```

### 2. Configure API
- Click "Configure API" on login page
- Enter: `http://localhost:8000/api`
- Click "Save"

### 3. Test Features
- Register account
- Login
- Browse courses
- Enroll in courses
- View dashboard

### 4. Use Postman
- Import collection from `postman/` directory
- Import environment
- Run requests
