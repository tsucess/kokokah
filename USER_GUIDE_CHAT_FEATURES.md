# User Guide: Chat Features

## Message Management

### Edit a Message
1. **Click on your message** in the chat
2. **Context menu appears** below the message
3. Click **"Edit"** button
4. **Edit modal opens** with your message text
5. **Modify the content** as needed
6. Click **"Save"** to update
7. Message updates with **(edited)** indicator

### Delete a Message
1. **Click on your message** in the chat
2. **Context menu appears** below the message
3. Click **"Delete"** button
4. **Confirmation modal appears**
5. Click **"Confirm"** to delete
6. Message is deleted from chat

**Note:** Only message owner or admins can edit/delete messages

## File Attachment

### Send a File
1. Click the **ðŸ“Ž paperclip icon** in the message input
2. **File picker opens**
3. **Select a file** from your device
4. **File preview modal shows:**
   - File icon
   - File name
   - File size
   - File type
5. Click **"Send File"** to send OR **"Cancel"** to close
6. **File appears in chat** as a downloadable file

**Supported File Types:**
- Documents: PDF, Word, Excel, PowerPoint, Text
- Images: JPG, PNG, GIF, WebP
- Archives: ZIP, RAR, 7Z

**File Size Limit:** 50MB

## Tips & Tricks

### Camera Tips
- Use good lighting for better photos
- Hold camera steady
- Use retake option if needed
- Photos are saved as JPEG

### File Tips
- Use descriptive file names
- Compress large files before sending
- Check file size (max 50MB)
- Supported formats shown in picker

### General Tips
- Click message to edit/delete
- Menu appears below message
- Click outside menu to close
- Works on mobile and desktop

## Troubleshooting

### Camera Not Working
- Check browser permissions
- Ensure camera is connected
- Try a different browser
- Check browser console for errors

### File Not Uploading
- Check file size (max 50MB)
- Verify file type is supported
- Check internet connection
- Try a different file

### Menu Not Appearing
- Refresh the page
- Check browser console
- Try a different browser
- Clear browser cache
