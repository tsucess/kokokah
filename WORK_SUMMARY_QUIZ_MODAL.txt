================================================================================
API ENDPOINT CONSUMED
================================================================================

Endpoint: POST /api/lessons/{lessonId}/quizzes
Method: QuizApiClient.createQuiz(lessonId, quizData)
Authentication: Bearer token (automatic)
Response: { success: true, data: {...quiz} }

The endpoint is fully integrated and ready to use. Instructors can now create
quizzes directly from the course curriculum editor with a user-friendly modal.

================================================================================
HOW IT WORKS
================================================================================

1. Instructor opens course editor
2. Expands topic to see lessons
3. Clicks "Add Quiz" button (question-circle icon) on lesson
4. Quiz modal opens with form
5. Instructor fills in:
   - Question text
   - Question type (MCQ or Alternative)
   - Options (2-4 depending on type)
   - Correct answer
   - Assigned marks
6. Instructor clicks "Add Category" button
7. Form validates required fields
8. QuizApiClient.createQuiz() called
9. API creates quiz in database
10. Success notification shown
11. Modal closes
12. Curriculum reloads
13. New quiz appears in curriculum
